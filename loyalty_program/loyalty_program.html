Відразу після реєстрації у вас будуть відображатись ціни для постійних покупців.


Про програму лояльності:

  • для деяких товарних позицій ціни для постійних покупців можуть не відрізнятися від стандартних цін;
  • у разі повернення товару покупцем, йому буде повернуто ту суму, за яку він придбав товар;
  • адміністрація сайту залишає за собою право змінювати умови програми лояльності будь-якої миті без попереднього повідомлення учасників програми лояльності.
